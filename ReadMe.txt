************ HOW TO INSTALL THE SYSTEM FOLLOW THE INSTRUCTIONS BELOW  ********************
Installation Procedure

1.Insert the installation CD in the DVD/CD ROM drive.

2.Install the xampp server or wampp server and MySQL server in your system. Follow the installation process as guided by the wizard.

3.Import the .sql files in the mysql server you have just installed.(.SQL FILE NAMED nokras(3) ) you will have to create a database called nokras.

4.Place the nokras folder at the xampp installation folder inside htdocs, for Wampp or xampp place it inside the www folder.

5.Now open up your Firefox or chrome browser and go to the universal resource locator of your browser and 
          type http://localhost/nokras. After typing the URL press go to launch the Nokras hotel website web application. 
            This is the customer view.

6. For the administrator. open up your Firefox or chrome browser and go to the universal resource locator 
           of your browser and type http://localhost/nokras/dashboard.php.
      After typing the URL press go to launch the Nokras hotel admin dashboard interface web application.
         In the interface there are various actions administrator can do.
